openxava / 文档 / 课程

本课程宗旨是教您如何使用 OpenXava 和其它 Java 相关技术、工具及框架开发 Java Enterprise 应用程序。我们将一起一步一步从头开发一个完整的应用程序。
OpenXava 的参考指南里描述了 OpenXava 所有语法和相关语义。它是必不可少的工具，可让您了解所有 OpenXava 的秘密。不过，如果您在 Java Enterprise 领域是新手，您需要的不仅是参考文档，您会想要更多示例代码并学习开发高级应用程序所需的相关技术如和方法。
在本课程您不仅会学到 OpenXava 还会学到 JPA、JUnit、Bean Validation、等。但最重要的，是您会学习在开发业务应用中面临的常见问题和更高级的要求的技术。
如果您对课程有任何问题，随时可以在帮助论坛中提问，若有任何建议可以到公开讨论论坛。
