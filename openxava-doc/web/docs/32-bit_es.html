openxava / documentación / 32 bits

Si estás usando una versión de OpenXava anterior a la 7.0 mira las instrucciones antiguas
OpenXava Studio (openxava.exe en Windows o el ejecutable openxava en Mac/Linux) requiere una máquina de 64 bits para funcionar. Sin embargo, puedes usar OpenXava en un ordenador de 32 bits, incluso la última versión. Sólo has de descargar un Java de 32 bits y usar Maven. Puedes usar cualquier editor de texto plano o si prefieres usar un IDE, puedes descargar un Eclipse de 32 bits.

Instalar Java para 32 bits

Ve a la página de descarga del JDK 8 de Oracle, allí escoge una distribución con x86 en el nombre (o i586 en el nombre de archivo), como Linux x86 Compressed Archive o Windows x86:
Descárgalo, ejecútalo y sigue todos los pasos para instalar el JDK en tu ordenador.

Descargar e instalar Maven

Descarga la última versión de Maven de Apache e instálala en tu ordenador.

Crear tu primer proyecto OpenXava

Ahora puedes seguir la guía de primeros pasos con OpenXava usando Maven para crear tu primer proyecto de OpenXava en tu ordenador de 32 bits. No necesitas OpenXava Studio, puedes hacerlo con línea de órdenes y un editor de texto plano.

Usar Eclipse

En lugar de usar un editor de texto puedes usar una IDE de Java, como Eclipse. Eclipse 2018-09 fue el último con soporte de 32 bits, por tanto descarga Eclipse 2018-09 para Windows 32 o para Linux 32. Descomprime el archivo descargado y pulsa en el ejecutable eclipse que tienes dentro de la carpeta eclipse.Después usa la opciones de importación de Eclipse para incluir tu proyecto OpenXava en el workspace de Eclipse. Eclipse ofrece un soporte muy completo para trabajar con proyectos Maven.
