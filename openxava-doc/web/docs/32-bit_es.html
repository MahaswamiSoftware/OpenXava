openxava / documentación / 32 bits

OpenXava Studio (openxava.exe en Windows o el ejecutable openxava en Mac/Linux) requiere una máquina de 64 bits para funcionar. Sin embargo, puedes usar OpenXava en un ordenador de 32 bits, incluso la última versión. Sólo has de descargar un Java de 32 bits y un Eclipse de 32 bits, después cambia el workspace de Eclipse para que apunte al incluido en la distribución de OpenXava.

Instalar Java para 32 bits

Ve a la página de descarga del JDK 8 de Oracle, allí escoge una distribución con x86 en el nombre (o i586 en el nombre de archivo), como Linux x86 Compressed Archive o Windows x86:
Descárgalo, ejecútalo y sigue todos los pasos para instalar el JDK en tu ordenador.

Descargar e instalar OpenXava

Descarga la última versión de OpenXava de openxava.org y descomprímela en tu ordenador. Tendrás una carpeta llamada openxava-6.x con tres subcarpetas: doc, studio y workspace. Vamos a usar la carpeta workspace en la siguiente sección. También hay un openxava.exe u openxava ejecutable de Linux en la raíz, no lo pulses, no funciona con 32 bits.

Descargar e instalar Eclipse

Eclipse 2018-09 fue el último con soporte de 32 bits, por tanto descarga Eclipse 2018-09 para Windows 32 o para Linux 32. Descomprime el archivo descargado y pulsa en el ejecutable eclipse que tienes dentro de la carpeta eclipse.
