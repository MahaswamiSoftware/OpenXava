Configurar entorno Visual Studio Code

Si aún no has instalado algún extensión para Java en Visual Studio Code, puedes visitar aquí y seguir las instrucciones de la sección Install Visual Studio Code for Java.

Crear un proyecto OpenXava nuevo

Crea un nuevo proyecto Maven con OpenXava, escogiendo org.openxava:openxava-archetype-spanish como arquetipo:

Empaqueta y ejecuta tu proyecto

Antes de ejecutar por primera vez tu proyecto, has de hacer un mvn package en él. Después en el explorador > Java Projects, ejecuta tuaplicacion o busca tuaplicacion.java (es el nombre de tu aplicación) dentro del paquete com.example.tuaplicacion.run en src/main/java, y ejecutala.

También puedes ver la Lección 1 de primeros pasos usando Visual Studio Code.
