Multiempresa/multimunicipio

La ingeniería inversa o retroingeniería es el proceso llevado a cabo con el objetivo de obtener información o un diseño a partir de un producto, en el caso de ingenieria inversa de un modelo desde una base de datos existente, nos referimos a obtener las entidades que mapean las tablas de la base de datos.
En este articulo, usaremos Dali de Eclipse para generar las entidades de una base de datos simple y adaptarlo para ser usado en OpenXava
