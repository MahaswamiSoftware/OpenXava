openxava / documentación / Aplicaciones de OpenXava como programas de escritorio

OpenXava genera una aplicación web diseñada para ser desplegada en un servidor de aplicaciones, sin embargo, con algunas adaptaciones puedes distribuir tu aplicación como un programa de escritorio independiente.
El próximo proyecto de GitHub demuestra cómo hacer eso:
https://github.com/vyshenskyy/DesktopOpenXava
